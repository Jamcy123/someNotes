# 笔记

剩余字符串小于 `k` 则全反转，否则反转 `k` 个，每次遍历移动 `2*k` 。

思路很简单，但把代码写简洁还是有难度。

```javascript
var reverseStr = function(s, k) {
    const len = s.length;
    s = [...s];
    for(let i = 0; i < len; i += 2*k) {
        let l = i - 1;
        let r = i + k < len ? i + k : len; // 确定需要反转的范围
        while(++l < --r) [s[l], s[r]] = [s[r], s[l]]; // 反转字符串
    }
    return s.join('');
};
```
